Crosswalk Project 10 Beta is available

Wed, 12 Nov 2014 12:00:00 GMT by Francesco Balestrieri

We are happy to announce that Crosswalk Project 10 for Android has entered the Beta Channel.

This release is rebased to the latest Chromium Beta (39) and introduces the following new features:

Updated Embedding API (v4), adding the following methods:

  • getRemoteDebuggingUrl
  • onReceivedSslError
  • onCreateWindowRequested
  • onIconAvailable and onReceivedIcon

Support for Cordova 3.6.3

Crosswalk Project AAR package in Stable and Beta channels for Maven/Gradle integration (documentation)

Remote debugging is disabled by default and needs to be enabled with --enable-remote-debugging

Full release notes in Jira

Current release blockers to be resolved before promotion to stable

You can propose new release blockers by adding the Release-Blocker label to a bug.
