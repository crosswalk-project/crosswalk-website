The Crosswalk Project would like to highlight applications that use its runtime.

The information provided as part of this form will be used to display the application on the Crosswalk Project website. The application will likely look similar to the 'Display Preview' shown with this form, but may also show additional information submitted here. The application will be displayed with other applications. These may or may not be comptetitors' applications.

If the Play Store URL is provided, it will be used to link the application icon to the store website. If the Author URL is provided, it will be used to link the Author's name to that website.

Once submitted, a confirmation email will be sent to the email address provided in the form.

Removal
Your application can be removed from the Crosswalk Project website at any time. The removal process is explained in the confirmation email you receive upon submission. All efforts will be made to be responsive to this request. Upon removal, all submitted information, including email address, will be deleted.

Use of Email Address
Your email address will only be used to contact you regarding this application submission and listing on crosswalk-project.org. Intel will only use this information for the purpose in which it was collected and will not share or repurpose this information without first securing your consent. Intel reserves the right to contact you periodically regarding the application listing such as if the application is no longer available in the application store, is no longer maintained, or should be removed from our site.

Intel respects your privacy. To learn more about Intel’s privacy practices, please go to www.intel.com/privacy.
