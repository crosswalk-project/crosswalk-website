Samples

We have packaged some simple applications to get you up and running with Crosswalk. These applications are available as a download. The source code is on github (MIT licence).

To use the samples, download the sample archive, then unpack it from the command line:

tar xvf crosswalk-samples-0.8.tar.gz

This will create a crosswalk-samples-0.8 directory with several project sub-directories:

  • extensions-android – A simple Crosswalk Android extension example. The Android extensions tutorial explains how this was developed. See this README for build instructions.

  • hello_world – A minimal example.

  • simd-mandelbrot – An example of using Crosswalk's native support for SIMD.

  • space-dodge-game – A game where you dodge asteroids in a spaceship. This contains the code developed for the article about Crosswalk's screen APIs.

  • tizen_apis – A sample application using the Tizen APIs.

  • webgl – An example of how to use WebGL in Crosswalk via three.js.

  • webrtc – An example of using the WebRTC API in Crosswalk. This is the example developed in the WebRTC tutorial. See this README for build instructions.

For more information about the samples, follow the links in the menu.

Follow @xwalk_project on Twitter for hybrid app development resources.
