The Crosswalk Architecture

Crosswalk provides an HTML5 application runtime built using various other open source projects.

Crosswalk extends the features available on the web platform today with new capabilities for applications written using web technologies. This includes the introduction of an application Runtime Model, an implementation of the W3C SysApps Working Group APIs, and a JavaScript Extension Framework which allows application vendors to add additional native capabilities (if necessary.)

At the heart of Crosswalk is the Blink rendering and layout engine. This provides the same HTML5 features and capabilities expected in the modern web ecosystem.

Building on Blink, Crosswalk uses the Chromium Content module to provide a multi-process architecture. The Chromium Content module is designed for security and performance. In addition to interacting with Blink, the Content Module manages the Graphics Subsystem. The Graphics Subsystem provides 2D rendering services and Web GL.
